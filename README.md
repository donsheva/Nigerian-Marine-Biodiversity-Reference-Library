# Nigerian-Marine-Biodiversity-Reference-Library
An open-access marine biodiversity data and analytics platform providing environmental DNA (eDNA) reference libraries, species occurrence records, and computational tools for ecological monitoring and conservation planning in Nigerian coastal, lagoon ecosystems and the Gulf of Guinea.

The Nigeria Marine Biodiversity & eDNA Data Platform is an open-access scientific repository developed to support marine ecological research, biodiversity conservation, and environmental monitoring across Nigerian coastal and lagoon ecosystems. It is a systematic initiative to build the first comprehensive, voucher-linked molecular barcode database for the coastal and marine species of the Gulf of Guinea. This project directly addresses a critical gap in global environmental DNA (eDNA) metabarcoding infrastructure for West African ecosystems.

**Project Mission**
A major limitation of eDNA metabarcoding in West Africa is the lack of regionally representative reference sequences. This library provides validated sequences from dominant lagoon and marine taxa to improve taxonomic assignment accuracy for future biodiversity surveys and support long-term conservation planning in Nigeria.

**🎯 Objectives**
Develop a centralized marine biodiversity database for Nigeria
Support environmental DNA (eDNA) research and biomonitoring
Enable species identification using genetic reference libraries
Provide open tools for ecological data analysis
Strengthen conservation planning and marine policy decisions

**Key Features**
Taxonomic Scope: Coastal and estuarine fish, macrobenthos (gastropods, bivalves), and meiofauna.

**Geographic Scope**: Nigerian Coastal Waters, Lagoons, Estuaries, and Offshore Marine Ecosystems, Gulf of Guinea, West Africa (Ghana, Cameroon, Benin, Togo, Senegal, Guinea, Côte d'Ivoire, Liberia).

**Methodology**: All sequences are generated from physical voucher specimens that are photographed and morphologically identified.
